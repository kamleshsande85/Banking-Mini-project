# Banking Application

## Overview
A simple banking app for account management and admin oversight.

## Features
- User: Create accounts, deposit/withdraw, check balance, view history.
- Admin: Manage users, monitor transactions.
- SQLite for storage.

## Requirements
- Python 3.x
- Tkinter (built-in)
- SQLite (built-in)

## Running the Application
1. Clone the repo.
2. Navigate to the project directory.
3. Run `python app.py`.
